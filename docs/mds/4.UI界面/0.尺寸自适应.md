### UI对象可以使用自适应来自动适配分辨不同的屏幕

```lua
--- 未开启自适应
FrameBackdrop(kit .. "->main", FrameGameUI)
    :relation(FRAME_ALIGN_CENTER, FrameGameUI, FRAME_ALIGN_CENTER, 0, 0)
    :size(0.1, 0.1)
    :texture("bg")
```

```lua
--- 已开启自适应
FrameBackdrop(kit .. "->main", FrameGameUI)
    :relation(FRAME_ALIGN_CENTER, FrameGameUI, FRAME_ALIGN_CENTER, 0, 0)
    :size(0.1, 0.1)
    :texture("bg")
    :adaptive(true)
```
